# Cookie icon SVG variations

This folder contains simple SVG options for replacing the existing cookie-in-a-trashcan artwork.

- `cookie-red-x.svg`: filled cookie with a bold red X.
- `cookie-red-x-outline.svg`: filled cookie with a white-backed red X for stronger contrast.
- `cookie-prohibited.svg`: filled cookie with a red prohibition ring and slash.
- `cookie-prohibited-bold.svg`: filled cookie with a thicker prohibition mark and slightly larger cookie.
- `cookie-prohibited-flat.svg`: flatter cookie without a dark outer stroke.
- `cookie-prohibited-offset.svg`: cookie shifted behind the centered prohibition mark.
- `cookie-prohibited-dark.svg`: darker cookie with a white-backed prohibition mark.
- `cookie-prohibited-badge.svg`: cookie and prohibition mark on a pale circular backing.
- `cookie-prohibited-outline.svg`: filled cookie with a white-backed prohibition mark for visibility on busy backgrounds.
- `cookie-prohibited-mono.svg`: minimal one-color cookie with a red prohibition mark for very small sizes.

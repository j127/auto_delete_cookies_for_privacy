# Permissions

When you install ADCP, the browser lists the permissions it wants. Here is what each one is actually for. (The same table, in policy form, lives in the repository's PRIVACY.md — the two are kept in sync.)

| Permission                                                | Why ADCP needs it                                                                                                                                                                                                                            |
| --------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Read and change site data on all sites** (`<all_urls>`) | The scary-sounding one, and the unavoidable one: cookies can belong to any site you visit, so deleting them requires access to all sites. ADCP uses it exclusively to read and remove cookies and site data — it does not read page content. |
| Cookies                                                   | The core job: list a site's cookies (for the badge count) and delete them.                                                                                                                                                                   |
| Browsing data                                             | Delete the optional extras: cache, IndexedDB, LocalStorage, plugin data, service workers.                                                                                                                                                    |
| Storage                                                   | Save your settings and keep lists locally, plus the small working state that survives the extension's background process being put to sleep.                                                                                                 |
| Alarms                                                    | Schedule the delayed cleanup so it still fires if the browser suspends the extension's background process during the delay (a normal Manifest V3 behavior).                                                                                  |
| Tabs                                                      | Know which sites are open (open sites are protected) and notice when a tab closes or moves to another domain.                                                                                                                                |
| Active tab                                                | Act on the current tab for popup buttons without needing anything broader at click time.                                                                                                                                                     |
| Scripting                                                 | Run the on-demand "wipe LocalStorage for this site" action inside the page, since modern extensions can't inject inline scripts.                                                                                                             |
| Context menus                                             | The optional right-click menu. Can be switched off in settings.                                                                                                                                                                              |
| Notifications                                             | The optional "cleaned X sites" notifications. Also switchable.                                                                                                                                                                               |

## What's deliberately absent

No permission for browsing history, no web request interception, no external connectivity of any kind. The extension cannot see what you type, which pages you read, or anything inside a page — and it never talks to a server, because it doesn't have one.

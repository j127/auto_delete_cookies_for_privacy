Design notes — 05d · Shield / Simple popup

Direction: the trust dashboard taken to its logical simple end. The popup's default state has no expressions, no wildcard syntax — one primary "Protect this site" whose caption states the honest default (whitelists *.github.com, i.e. the site including subdomains, which is what non-technical users mean), a greylist equivalent in plain words ("Protect until restart"), and Clean now. The gate is a real setting: "Show advanced controls in the popup" in Settings → Interface — in this mockup the checkbox actually toggles the popup frame above via CSS :has(), exactly the interaction the real extension would implement. Discoverability is handled by the popup's quiet "More controls…" footer link, which would deep-link to this settings row. Palettes: Guardian (default), Signal, Bakery; cookie mark as in 05c. Precedent for the pattern: uBlock Origin's "I am an advanced user" checkbox.

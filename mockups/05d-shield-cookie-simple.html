Design notes — 05d · Shield / Simple popup (iteration 3)

Final copy: hero resting state is "Auto-delete is on / This site won't remember you." and it swaps to "Keeping this site's cookies / github.com is on your keep list" when a keep rule matches — the pair is shown in the Hero-states strip. Primary action is final: "Keep cookies for this site". Naming: the whitelist/greylist jargon never reaches the UI — one verb everywhere: Keep (permanent) and Keep this session (deleted after the browser restarts), with hover tooltips carrying the full explanations; "keep list" is the user-facing name. Clean actions are scope-honest: "Clean now" (skips kept sites and open tabs — tooltip says so), "Clean now, including open tabs", "Delete this site's cookies", "Delete everything this site stored". Name bar: logo removed (the hero mark below already carries it; the settings sidebar keeps its copy), and a quiet Share dropdown links to the extension's page — the GitHub repo until publication, the Chrome Web Store listing afterwards (tracked by a standing issue). Toggles follow --c-accent. Inspector tabs: advanced mode gains Overview / Cookies / Storage (pure CSS radio tabs) — the design for the #112 site-data inspector and its #114 popup panel.
